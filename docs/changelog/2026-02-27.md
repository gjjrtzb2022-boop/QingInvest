# 2026-02-27 详细更新日志

## 更新主题
全站视觉重构 + 文章详情页沉浸阅读升级。

## 背景说明
本轮迭代聚焦两条主线：

1. 全站视觉与主题体系（白昼/夜幕）的统一和可读性强化。
2. 文章详情页结构、交互与阅读体验的深度打磨，向目标参考站点靠齐。

---

## 详细变更

### 1) 文章详情页架构重做
- 详情页重构为三栏模式：左侧「文章系列」、中间正文、右侧目录与批注。
- 中间主区信息层级重排：面包屑、标题、摘要、元信息、阅读状态、标签、系列卡、来源信息、正文、相关阅读、翻页。
- 移除右侧「投资体系」卡片，降低干扰，让注意力更聚焦在正文和目录。

### 2) 侧栏收展与沉浸阅读
- 左侧「文章系列」和右侧「目录」标题区可直接点击收起/展开。
- 当侧栏收起后，在中间与边栏间隙显示可点击触发键，支持再次展开。
- 间隙触发键为 `sticky`，页面下滑时会跟随，避免“回到顶部才能展开”。
- 单侧收起时中间内容区自动加宽并放大排版；双侧收起时进入更强沉浸阅读状态。

### 3) 文案与交互统一
- 左侧标题统一由「文章目录」改为「文章系列」。
- 相关触发文案、提示文案、可访问性 label 一并同步调整。
- 折叠/展开交互从顶部按钮改为边栏内触发，减少视觉噪音。

### 4) 溢出与细节修复
- 左侧文章项标题超出边界时自动单行省略号显示，避免文本溢出破版。
- 优化右侧目录与卡片边界的内边距关系，使视觉更贴近参考样式。
- 修复「仅收起左侧」时的布局异常，避免网格错位。

### 5) 版式与留白节奏
- 文章详情页左右外留白显著收紧，提升有效阅读宽度。
- 保留顶部呼吸区，避免内容贴近导航，整体观感更稳。
- 多轮微调字号、权重、边距和卡片比例，提升正文可读性。

### 6) 主题模式与可读性
- 白昼/夜幕模式切换持续优化，状态保存到 `localStorage`。
- 夜幕模式强化为黑灰高对比风格。
- 修复深色场景下标题、正文、目录、按钮与弹窗文字可读性问题。

### 7) 首页装饰与视觉氛围
- 两侧装饰从旧元素升级为元宵节主题挂饰。
- 保留不可选中、不可误触属性，避免影响阅读交互。
- 发光与摆动动效节奏优化，避免过度抢焦点。
