# 2026-02-26 详细更新日志

## 更新主题
文章库迁移到新架构并上线首批阅读交互能力。

## 背景说明
本次更新聚焦「清一文章库」的工程化与可持续维护，目标是让后续内容导入、校验、展示、检索形成稳定闭环。

---

## 详细变更

### 1) 架构与发布能力
- 网站从旧静态形态迁移到 Next.js + TypeScript + Tailwind 工程架构。
- 建立静态导出流程，适配 GitHub Pages 发布。
- 增加 CI 基础校验：文章缺少关键字段（title/date/slug）时阻止发布。

### 2) 文章库核心能力
- 列表页支持分页（每页 20 篇）与 URL 参数化筛选。
- 支持按状态、系列、分类、标签检索，并支持按日期/标题排序。
- 详情页支持同系列导航、相关内容推荐、上一篇/下一篇切换。

### 3) 内容生产效率
- 新增批量导入脚本，可将原文快速生成 Markdown 初稿。
- 新增文章索引构建脚本与内容校验脚本，降低手工维护成本。
- 文章数据统一采用 Markdown 文件管理，便于扩展与版本管理。

### 4) 阅读交互首版
- 正文选中后增加悬浮操作条：批注 / 金句 / 复制 / 分享 / 搜索。
- 批注与金句同步写入右侧批注面板（浏览器本地存储）。
- 完成基础字号、字重、配色、间距微调，提升阅读稳定性。

---

## 本次结果
- 文章库从内容到工程链路已打通，可稳定迭代。
- 后续重点可转向：视觉一致性提升与文章内容规模化建设。
